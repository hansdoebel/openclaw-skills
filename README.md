# openclaw-skills

A collection of [OpenClaw skills](https://docs.openclaw.ai) for integrating with external APIs, services, and workflows.

## Skills

| Skill | Description |
|-------|-------------|
| [bewerbungsanschreiben](./bewerbungsanschreiben/SKILL.md) | Professionelle deutsche Bewerbungsanschreiben nach DIN 5008 als PDF via Typst |
| [jina-ai](./jina-ai/SKILL.md) | Web reading, search, and content extraction powered by Jina AI APIs |
| [qdrant](./qdrant/SKILL.md) | Semantic memory with Qdrant vector database and Jina AI embeddings |
| [random-quote](./random-quote/SKILL.md) | Random inspirational quotes |

## Usage

Each skill is defined by a `SKILL.md` file that teaches the agent how to use the corresponding tool or API. To use a skill, copy its folder into your workspace's `skills/` directory or install it from [ClawhHub](https://clawhub.com).
